4.10 — Antiderivatives

We’ve spent a lot of time differentiating functions. We now ask the opposite question: if we start with a derivative, can we find the original function?

The reverse of differentiation

A function FFF is an antiderivative of the function fff if
F′(x)=f(x)F'(x) = f(x)F′(x)=f(x)

for all xxx in the domain of fff.

For example, let f(x)=3x2f(x)=3x^2f(x)=3x2. Find the antiderivative of f(x)f(x)f(x).

If our function has a power of 222, then we know that its antiderivative must have a power of 2+12+12+1 from the power rule.

So far, we have that the antiderivative of fff is F(x)=x3F(x)=x^3F(x)=x3.

And using the definition, we see that
F(x)=x3  ⟹  F′(x)=3x2.F(x) = x^3 \implies F'(x) = 3x^2.F(x)=x3⟹F′(x)=3x2.

However, we cannot conclude that F(x)=x3F(x)=x^3F(x)=x3 is the antiderivative of fff. This is because there may be a constant in the antiderivative that produces the same function. For example, x3+1x^3+1x3+1 is also an antiderivative of 3x23x^23x2, and so is x3+πx^3 + \sqrt{\pi}x3+π​.
ddx(x3+1)=ddx(x3+π)=3x2\diff{}{x}(x^3+1)=\diff{}{x}(x^3+\sqrt{\pi})=3x^2dxd​(x3+1)=dxd​(x3+π​)=3x2

As we can see, since the derivative of any constant ccc is zero, we must consider that in our expression for the antiderivative.

As such, we conclude that the antiderivative of f(x)=3x2f(x)=3x^2f(x)=3x2 is F(x)=x3+cF(x)=x^3+cF(x)=x3+c.

Let f(x)=4xf(x)=4xf(x)=4x. Find the antiderivative F(x)F(x)F(x).

f(x)=4x  ⟹  F(x)=2x2+cf(x) = 4x \implies F(x) = 2x^2 + cf(x)=4x⟹F(x)=2x2+c

Again, to check our assumption, we can differentiate FFF where ccc is some constant.
F(x)=2x2+c  ⟹  F′(x)=4xF(x) = 2x^2 + c \implies F'(x) = 4xF(x)=2x2+c⟹F′(x)=4x

As we can see, differentiating the antiderivative gives us fff.

As such, the antiderivative is F(x)=2x2+cF(x)=2x^2+cF(x)=2x2+c.

Let f(x)=7x3f(x) = 7x^3f(x)=7x3. Find the antiderivative F(x)F(x)F(x).

f(x)=7x3  ⟹  F(x)=74x4+cf(x) = 7x^3 \implies F(x)=\frac{7}{4}x^4+cf(x)=7x3⟹F(x)=47​x4+c

Reverse power rule

We can apply the power rule of differentiation backwards in order to find its antiderivative.

If fff is a function such that f(x)=axnf(x)=ax^nf(x)=axn, where aaa and nnn are constants such that n≠−1n\neq-1n=−1.

Then, the antiderivative of fff is
F(x)=axn+1n+1+cF(x) = \frac{ax^{n+1}}{n+1}+cF(x)=n+1axn+1​+c

where ccc is a constant.

Let f(x)=−8x3f(x)=-8x^3f(x)=−8x3. Find the antiderivative F(x)F(x)F(x).

Using the reverse power rule, we have that
F(x)=−8x3+13+1+c=−8x44+c=−2x4+c.\begin{aligned} F(x) &= \frac{-8x^{3+1}}{3+1}+c \\[1em] &= \frac{-8x^4}{4}+c \\[1em] &= -2x^4+c. \end{aligned}F(x)​=3+1−8x3+1​+c=4−8x4​+c=−2x4+c.​

Let f(x)=xf(x) = \sqrt{x}f(x)=x​. Find the antiderivative F(x)F(x)F(x).

Rewrite x\sqrt{x}x​ and use the reverse power rule.
f(x)=x=x12F(x)=x3232+c=23x32+c=23x3+cf(x) = \sqrt{x} = x^\frac{1}{2}\\ \begin{aligned} F(x) &= \frac{x^\frac{3}{2}}{\frac{3}{2}} + c \\[1.3em] &= \frac{2}{3}x^\frac{3}{2} + c \\[1em] &= \frac{2}{3}\sqrt{x^3} + c \end{aligned}f(x)=x​=x21​F(x)​=23​x23​​+c=32​x23​+c=32​x3​+c​

Let f(x)=4x2f(x) = \frac{4}{x^2}f(x)=x24​. Find the antiderivative F(x)F(x)F(x).

f(x)=4x2=4x−2F(x)=4x−1−1+c=−4x−1+c=−4x+cf(x) = \frac{4}{x^2} = 4x^{-2} \\ \begin{aligned} F(x) &= \frac{4x^{-1}}{-1}+c \\[1em] &= -4x^{-1}+c \\[.5em] &= -\frac{4}{x}+c \end{aligned}f(x)=x24​=4x−2F(x)​=−14x−1​+c=−4x−1+c=−x4​+c​

Other nasty antiderivatives

Let f(x)=sin⁡xf(x)=\sin xf(x)=sinx. Find the antiderivative.

f(x)=sin⁡x  ⟹  F(x)=−cos⁡(x)+cf(x)=\sin x \implies F(x) = -\cos(x)+cf(x)=sinx⟹F(x)=−cos(x)+c

Let f(x)=5sec⁡2(x)f(x)=5\sec^2(x)f(x)=5sec2(x). Find the antiderivative.

f(x)=5sec⁡2(x)  ⟹  F(x)=5tan⁡x+cf(x)=5\sec^2(x) \implies F(x)=5\tan x + cf(x)=5sec2(x)⟹F(x)=5tanx+c

Let f(x)=exf(x)=e^xf(x)=ex. Find the antiderivative.

f(x)=ex  ⟹  F(x)=ex+cf(x) = e^x \implies F(x) = e^x + cf(x)=ex⟹F(x)=ex+c

Let f(x)=x−1f(x) = x^{-1}f(x)=x−1. Find the antiderivative.

Note that we cannot apply the reverse power rule here.
F(x)=x−1+1−1+1=x00=10F(x)=\frac{x^{-1+1}}{-1+1}=\frac{x^0}{0}=\frac{1}{0}F(x)=−1+1x−1+1​=0x0​=01​

This is a special case where we cannot apply the reverse power rule (where the power is −1-1−1).

Instead, recall that ddxln⁡x=1x\diff{}{x}\ln x = \frac{1}{x}dxd​lnx=x1​.
f(x)=x−1=1x  ⟹  F(x)=ln⁡∣x∣+cf(x) = x^{-1} = \frac{1}{x} \implies F(x) = \ln |x| + cf(x)=x−1=x1​⟹F(x)=ln∣x∣+c

Notational fuckery

There is a universal notation used for antiderivatives. It is known as the indefinite integrals, and it uses a symbol called an integral sign.

Given a function fff, the indefinite integral of fff, denoted
∫f(x) ⁣dx\int f(x)\dd x∫f(x)dx
is the most general antiderivative of fff.

If FFF is an antiderivative of fff, then
∫f(x) ⁣dx=F(x)+c.\int f(x)\dd x = F(x) + c.∫f(x)dx=F(x)+c.

  • The expression f(x)f(x)f(x) is called the integrand.
  • xxx is called the variable of integration.
  • And the expression ∫f(x) ⁣dx\int f(x)\dd x∫f(x)dx is called "the indefinite integral of fff."

Compute ∫2x ⁣dx\int2x\dd x∫2xdx.

Again, all this means is that it is asking for the antiderivative of 2x2x2x.
∫2x ⁣dx=2x22+c=x2+c\int2x\dd x = \frac{2x^2}{2} + c = x^2 + c∫2xdx=22x2​+c=x2+c

Compute ∫(x3+5x2+ex) ⁣dx\int(x^3+5x^2+e^x)\dd x∫(x3+5x2+ex)dx.

Just like when we differentiate a sum, we can integrate them separately — but don’t forget the +c+c+c!
∫(x3+5x2+ex) ⁣dx=x44+5x33+ex+c=14x4+53x3+ex+c\begin{aligned} \int(x^3+5x^2+e^x)\dd x &= \frac{x^4}{4}+\frac{5x^3}{3}+e^x+c \\ &= \frac{1}{4}x^4 + \frac{5}{3}x^3 + e^x + c \end{aligned}∫(x3+5x2+ex)dx​=4x4​+35x3​+ex+c=41​x4+35​x3+ex+c​

With the new fancy notation, we can rewrite our reverse power rule.

Power rule for integrals

For n≠−1n\neq-1n=−1,
∫xn ⁣dx=xn+1n+1+c\int x^n \dd x = \frac{x^{n+1}}{n+1}+c∫xndx=n+1xn+1​+c

where ccc is a constant.

Initial-value problems

Normally, when finding an antiderivative, we include a ccc to indicate an unknown constant. But, if we have additional information about our function, we can figure out what these constants are.

Let f′(x)=x−3f'(x)=x^{-3}f′(x)=x−3. Find f(x)f(x)f(x) where f(1)=1f(1)=1f(1)=1.

f′(x)=x−3  ⟹  f(x)=−12x2+cf'(x)=x^{-3} \implies f(x) =-\frac{1}{2x^2}+cf′(x)=x−3⟹f(x)=−2x21​+c

Since we are given that f(1)=1f(1)=1f(1)=1, we can use this to solve for ccc.
−12(1)2+c=1∴c=32-\frac{1}{2(1)^2}+c=1 \\ \therefore c = \frac{3}{2}−2(1)21​+c=1∴c=23​

As such, we have that
f(x)=−12x2+32.f(x) = -\frac{1}{2x^2}+\frac{3}{2}.f(x)=−2x21​+23​.
