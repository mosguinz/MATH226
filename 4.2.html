Refresher: tangent lines

Previously, we have written equations for tangent lines of functions at specific points.

Equation of a tangent line

The equation of the tangent to f(x)f(x)f(x) at the point x=ax = ax=a is given by:
y=f(a)+f′(a)(x−a).y = f(a) + f'(a)(x-a).y=f(a)+f′(a)(x−a).

Let f(x)=1xf(x) = \frac{1}{x}f(x)=x1​. Find the equation of the tangent line at x=2x = 2x=2.

f(2)=12∴(2,f(2))=(2,12)f(2) = \frac{1}{2} \\ \therefore (2, f(2)) = \(2, \frac{1}{2}\)f(2)=21​∴(2,f(2))=(2,21​)

f′(x)=−x−2=−1x2∴f′(2)=−14f'(x) = -x^{-2} = -\frac{1}{x^2} \\ \therefore f'(2) = -\frac{1}{4}f′(x)=−x−2=−x21​∴f′(2)=−41​

Linear approximation

While the function f(x)f(x)f(x) and its tangent only intersects at a single point, the neighborhood around that point the function and tangent line are "close." Due to this fact, we can use a tangent line to approximate a function’s value at a specific point.

Let’s continue to use the function f(x)=1xf(x) = \frac{1}{x}f(x)=x1​ as our example.
Below, f(x)f(x)f(x) is graphed in red, where its tangent line at x=2x = 2x=2 is in green.
We can see on the left that its tangent line at x=2x = 2x=2 (green) only touches the graph of fff (red) once at that point.
When we zoom in closer around (2,12)(2, \frac{1}{2})(2,21​), we can see that the neighboring values around 222 is very close to the value of f(2)f(2)f(2).

This process is known as linear approximation.

When performing linear approximation, we make a linear approximation function and use it to make predictions for f(x)f(x)f(x).

Linear approximation function

From our earlier example, we concluded that the equation of the tangent line to a differentiable function fff at x=ax = ax=a can be used to approximate f(x)f(x)f(x) for xxx near aaa.

f(x)≈f(a)+f′(a)(x−a)where x is near a.f(x) \approx f(a) + f'(a)(x - a) \quad \text{where }x\text{ is near }a.f(x)≈f(a)+f′(a)(x−a)where x is near a.

As such, we can derive the linear approximation function, LLL, to be the following:
L(x)=f(a)+f′(a)(x−a)L(x) = f(a) + f'(a)(x - a)L(x)=f(a)+f′(a)(x−a)

So, what’s the fucking point of this?

Linear approximations are and/or were used to estimates computations that are too difficult to calculate by hand, like 9.1\sqrt{9.1}9.1​.
