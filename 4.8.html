4.8 — L’Hôpital’s Rule

Recap: limit theory

Recall that if:
lim⁡x→af(x)=L1lim⁡x→ag(x)=L2≠0.\lim_{x\to a}f(x) = L_1 \\ \lim_{x\to a}g(x) = L_2 \neq 0.x→alim​f(x)=L1​x→alim​g(x)=L2​=0.

Then,
lim⁡x→af(x)g(x)=L1L2.\lim_{x\to a}\frac{f(x)}{g(x)} = \frac{L_1}{L_2}.x→alim​g(x)f(x)​=L2​L1​​.

But… what if L1,L2=0L_1, L_2 = 0L1​,L2​=0? Like
lim⁡x→2x2−4x−2.\lim_{x\to2}\frac{x^2-4}{x-2}.x→2lim​x−2x2−4​.

Evaluating the limits of the numerator and denominator yields zero for both.
lim⁡x→2(x2−4)=lim⁡x−2(x2−4)=0\lim_{x\to2}(x^2-4) = \lim_{x-2}(x^2-4) = 0x→2lim​(x2−4)=x−2lim​(x2−4)=0

Which then gives us an indeterminate form 00\frac{0}{0}00​. Previously, we learned that we can factor the expression in order to evaluate the limit.
lim⁡x→2x2−4x−2=lim⁡x→2(x−2)(x+2)x−2=4\lim_{x\to2}\frac{x^2-4}{x-2} = \lim_{x\to2}\frac{\cancel{(x-2)}(x+2)}{\cancel{x-2}} = 4x→2lim​x−2x2−4​=x→2lim​x−2​(x−2)​(x+2)​=4

However, we can’t do the same with expressions like
lim⁡x→0sin⁡xx\lim_{x\to0}\frac{\sin x}{x}x→0lim​xsinx​

because
lim⁡x→0sin⁡0=lim⁡x→0x=0\lim_{x\to0}\sin0 = \lim_{x\to0}x = 0x→0lim​sin0=x→0lim​x=0

and evaluating them by plugging them in would give us 00\frac{0}{0}00​, which is indeterminate.

We can get around this using L’Hôpital’s rule, which uses local linear approximations to evaluate limits that produces indeterminate forms lke the ones above and many more.

L’Hôpital’s rule

L’Hôpital’s rule (0/0\bold{0/0}0/0 form)

Suppose fff and ggg are differentiable functions over an open interval containing aaa, except possibly at aaa. If lim⁡x→af(x)=lim⁡x→ag(x)=0\displaystyle\lim_{x\to a} f(x) = \lim_{x\to a} g(x) = 0x→alim​f(x)=x→alim​g(x)=0, then

lim⁡x→af(x)g(x)=lim⁡x→af′(x)g′(x),\lim_{x\to a}\frac{f(x)}{g(x)} = \lim_{x\to a}\frac{f'(x)}{g'(x)},x→alim​g(x)f(x)​=x→alim​g′(x)f′(x)​,

assuming the limit on the right exists or is ∞\infty∞ or −∞-\infty−∞. This also holds if the limit is infinite where a=∞a=\inftya=∞ or a=−∞a=-\inftya=−∞, or the limit is one-sided.

Evaluate lim⁡x→ax2−4x−2\displaystyle\lim_{x\to a}\frac{x^2-4}{x-2}x→alim​x−2x2−4​.

First, check that L’Hôpital’s rule actually applies, i.e., that the limit of the numerator and denominator is zero.
lim⁡x→2(x2−4)=22−4=0=lim⁡x→2(x−2)=2−2\lim_{x\to2}(x^2-4) = 2^2-4 = 0 = \lim_{x\to2}(x-2) = 2-2x→2lim​(x2−4)=22−4=0=x→2lim​(x−2)=2−2

Since x2−4→0x^2-4\to0x2−4→0 and x−2→0x-2\to0x−2→0, we can apply L’Hôpital’s rule.
lim⁡x→2x2−4x−2=lim⁡x→2ddx(x2−4)ddx(x−2)=lim⁡x→22x1=4\begin{aligned} \lim_{x\to2}\frac{x^2-4}{x-2} &= \lim_{x\to2}\frac{\diff{}{x}(x^2-4)}{\diff{}{x}(x-2)} \\[1.5em] &= \lim_{x\to2}\frac{2x}{1} \\[1em] &= 4 \end{aligned}x→2lim​x−2x2−4​​=x→2lim​dxd​(x−2)dxd​(x2−4)​=x→2lim​12x​=4​

Now, let’s try using L’Hôpital’s rule to evaluate lim⁡x→0sin⁡xx\displaystyle\lim_{x\to0}\frac{\sin x}{x}x→0lim​xsinx​.

Again, checking if both sin⁡x→0\sin x\to0sinx→0 and x→0x\to0x→0.
lim⁡x→0(sin⁡x)=sin⁡0=0=lim⁡x→0x=0\lim_{x\to0}(\sin x) = \sin0 = 0 = \lim_{x\to0}x = 0x→0lim​(sinx)=sin0=0=x→0lim​x=0

Sure enough, they are. And so applying L’Hôpital’s rule gives us:
lim⁡x→0sin⁡xx=lim⁡x→0cos⁡x1=cos⁡0=1\lim_{x\to0}\frac{\sin x}{x} = \lim_{x\to0}\frac{\cos x}{1}=\cos0=1x→0lim​xsinx​=x→0lim​1cosx​=cos0=1

Thus, lim⁡x→0sin⁡xx=1\displaystyle\lim_{x\to0}\frac{\sin x}{x} = 1x→0lim​xsinx​=1. Much easier!

Evaluate lim⁡x→∞e1x−11x\displaystyle\lim_{x\to\infty}\frac{e^{\frac{1}{x}}-1}{\frac{1}{x}}x→∞lim​x1​ex1​−1​.

First, check if the rule applies.
lim⁡x→∞e1x−11x=1−10=00\lim_{x\to\infty}\frac{e^{\frac{1}{x}}-1}{\frac{1}{x}} = \frac{1-1}{0} = \frac{0}{0}x→∞lim​x1​ex1​−1​=01−1​=00​

It does, so we apply the rule.
lim⁡x→∞e1x−11x=lim⁡x→∞ddx(e1x−1)ddx(1x)=lim⁡x→∞−e1xx2−1x2=lim⁡x→∞x2e1xx2=lim⁡x→∞e1x=1\begin{aligned} \lim_{x\to\infty}\frac{e^{\frac{1}{x}}-1}{\frac{1}{x}} &= \lim_{x\to\infty}\frac{\diff{}{x}(e^{\frac{1}{x}}-1)}{\diff{}{x}\(\frac{1}{x}\)} \\[1.5em] &= \lim_{x\to\infty}\frac{-\frac{e^{\frac{1}{x}}}{x^2}}{-\frac{1}{x^2}} \\[1.5em] &= \lim_{x\to\infty}\frac{\cancel{x^2}e^\frac{1}{x}}{\cancel{x^2}} \\[1em] &= \lim_{x\to\infty}e^\frac{1}{x} \\ &= 1 \end{aligned}x→∞lim​x1​ex1​−1​​=x→∞lim​dxd​(x1​)dxd​(ex1​−1)​=x→∞lim​−x21​−x2ex1​​​=x→∞lim​x2x2ex1​​=x→∞lim​ex1​=1​

We can also use L’Hôpital’s rule when we get get an indeterminate form like ∞∞\frac{\infty}{\infty}∞∞​.

L’Hôpital’s rule (∞/∞\bold{\infty/\infty}∞/∞ form)

Suppose fff and ggg are differentiable functions over an open interval containing aaa, except possibly at aaa. If lim⁡x→af(x)=lim⁡x→ag(x)=∞\displaystyle\lim_{x\to a} f(x) = \lim_{x\to a} g(x) = \inftyx→alim​f(x)=x→alim​g(x)=∞ or lim⁡x→af(x)=lim⁡x→ag(x)=−∞\displaystyle\lim_{x\to a} f(x) = \lim_{x\to a} g(x) = -\inftyx→alim​f(x)=x→alim​g(x)=−∞, then

lim⁡x→af(x)g(x)=lim⁡x→af′(x)g′(x),\lim_{x\to a}\frac{f(x)}{g(x)} = \lim_{x\to a}\frac{f'(x)}{g'(x)},x→alim​g(x)f(x)​=x→alim​g′(x)f′(x)​,

assuming the limit on the right exists or is ∞\infty∞ or −∞-\infty−∞. This also holds if the limit is infinite where a=∞a=\inftya=∞ or a=−∞a=-\inftya=−∞, or the limit is one-sided.

Evaluate lim⁡x→∞3x+52x+1\displaystyle\lim_{x\to\infty}\frac{3x+5}{2x+1}x→∞lim​2x+13x+5​.

We could divide the numerator and denominator by xxx to find the limit. Or, we can use L’Hôpital’s rule.

First, check if L’Hôpital’s rule applies here.
lim⁡x→∞3x+52x+1  ⟹  lim⁡x→∞3x+5lim⁡x→∞2x+1=∞∞\lim_{x\to\infty}\frac{3x+5}{2x+1} \implies \frac{\displaystyle\lim_{x\to\infty}3x+5}{\displaystyle\lim_{x\to\infty}2x+1} = \frac{\infty}{\infty}x→∞lim​2x+13x+5​⟹x→∞lim​2x+1x→∞lim​3x+5​=∞∞​

Then, applying the rule, we have that:
lim⁡x→∞3x+52x+1=lim⁡x→∞ddx(3x+5)ddx(2x+1)=lim⁡x→∞32=32.\begin{aligned}\lim_{x\to\infty}\frac{3x+5}{2x+1} &= \lim_{x\to\infty}\frac{\diff{}{x}\(3x+5\)}{\diff{}{x}\(2x+1\)} \\[1.5em] &= \lim_{x\to\infty}\frac{3}{2} \\[1em] &= \frac{3}{2}. \end{aligned}x→∞lim​2x+13x+5​​=x→∞lim​dxd​(2x+1)dxd​(3x+5)​=x→∞lim​23​=23​.​

Evaluate lim⁡x→∞2x+311x2−7x+4\displaystyle\lim_{x\to\infty}\frac{2x+3}{11x^2-7x+4}x→∞lim​11x2−7x+42x+3​.

Clearly, the limits of both numerator and denominator is ∞\infty∞. As such, we apply the rule.

lim⁡x→∞2x+311x2−7x+4=lim⁡x→∞ddx(2x+3)ddx(11x2−7x+4)=lim⁡x→∞222x−7=0\begin{aligned} \lim_{x\to\infty}\frac{2x+3}{11x^2-7x+4} &= \lim_{x\to\infty}\frac{\diff{}{x}(2x+3)}{\diff{}{x}(11x^2-7x+4)} \\[1em] &= \lim_{x\to\infty}\frac{2}{22x-7} \\[1em] &= 0 \end{aligned}x→∞lim​11x2−7x+42x+3​​=x→∞lim​dxd​(11x2−7x+4)dxd​(2x+3)​=x→∞lim​22x−72​=0​

Evaluate lim⁡x→0ex−x−17x2\displaystyle\lim_{x\to0}\frac{e^x-x-1}{7x^2}x→0lim​7x2ex−x−1​.

Again, checking if the rule applies, we have that:
lim⁡x→0ex−x−17x2=lim⁡x→01−0−10=00.\lim_{x\to0}\frac{e^x-x-1}{7x^2} = \lim_{x\to0}\frac{1-0-1}{0} = \frac{0}{0}.x→0lim​7x2ex−x−1​=x→0lim​01−0−1​=00​.

Applying L’Hôpital’s rule, we have:
lim⁡x→0ex−x−17x2=lim⁡x→0ddx(ex−x−1)ddx(7x2)=lim⁡x→0ex−114x.\begin{aligned} \lim_{x\to0}\frac{e^x-x-1}{7x^2} &= \lim_{x\to0}\frac{\diff{}{x}(e^x-x-1)}{\diff{}{x}(7x^2)} \\[1em] &= \lim_{x\to0}\frac{e^x-1}{14x}. \end{aligned}x→0lim​7x2ex−x−1​​=x→0lim​dxd​(7x2)dxd​(ex−x−1)​=x→0lim​14xex−1​.​

If we try and evaluate the limit, we’ll find that the result we get is still indeterminate!
lim⁡x→0ex−114x=1−114⋅0=00\lim_{x\to0}\frac{e^x-1}{14x} = \frac{1-1}{14\cdot0} = \frac{0}{0}x→0lim​14xex−1​=14⋅01−1​=00​

As such, we can apply L’Hôpital’s rule again.
lim⁡x→0ex−114x=lim⁡x→0ex14=114\begin{aligned} \lim_{x\to0}\frac{e^x-1}{14x} &= \lim_{x\to0}\frac{e^x}{14} \\[1em] &= \frac{1}{14} \end{aligned}x→0lim​14xex−1​​=x→0lim​14ex​=141​​

And so, we have that lim⁡x→0ex−x−17x2=114\displaystyle\lim_{x\to0}\frac{e^x-x-1}{7x^2} = \frac{1}{14}x→0lim​7x2ex−x−1​=141​.

Using L’Hôpital’s rule for other indeterminate forms

L’Hôpital’s rule can also be used to evaluate expressions where you have other indeterminate forms, such as:
0⋅∞,∞−∞,1∞,00, and∞0.\begin{array}{c} 0\cdot\infty, & \infty-\infty, & 1^\infty, & 0^0,\text{ and} & \infty^0. \end{array}0⋅∞,​∞−∞,​1∞,​00, and​∞0.​

But, we must first manipulate the expression to be 00\frac{0}{0}00​ or ∞∞\frac{\infty}{\infty}∞∞​ in order to use L’Hôpital’s rule.

Evaluate lim⁡x→0+(x⋅ln⁡x)\displaystyle\lim_{x\to0^+}(x\cdot\ln x)x→0+lim​(x⋅lnx).

Using the limit laws, we have that:
lim⁡x→0+(x⋅ln⁡x)=lim⁡x→0+(x)lim⁡x→0+ln⁡(x)=0⋅∞.\lim_{x\to0^+}(x\cdot\ln x) = \lim_{x\to0^+}(x)\lim_{x\to0^+}\ln(x) = 0\cdot\infty.x→0+lim​(x⋅lnx)=x→0+lim​(x)x→0+lim​ln(x)=0⋅∞.

Here, we can’t apply L’Hôpital’s rule yet. However, we can manipulate it in such a way that one of the L’Hôpital’s rule cases applies.
lim⁡x→0+(x⋅ln⁡x)=lim⁡x→0+ln⁡x1x\lim_{x\to0^+}(x\cdot\ln x) = \lim_{x\to0^+}\frac{\ln x}{\frac{1}{x}}x→0+lim​(x⋅lnx)=x→0+lim​x1​lnx​

Then, we have that:
lim⁡x→0+ln⁡x1x=lim⁡x→0+ddx(ln⁡x)ddx(1x)=lim⁡x→0+1x−1x2 =lim⁡x→0+(−x)=0.\begin{aligned} \lim_{x\to0^+}\frac{\ln x}{\frac{1}{x}} &= \lim_{x\to0^+}\frac{\diff{}{x}(\ln x)}{\diff{}{x}\(\frac{1}{x}\)} \\[1.5em] &= \lim_{x\to0^+} \frac{\frac{1}{x}}{-\frac{1}{x^2}}\ \\[1em] &= \lim_{x\to0^+}(-x) \\ &= 0. \end{aligned}x→0+lim​x1​lnx​​=x→0+lim​dxd​(x1​)dxd​(lnx)​=x→0+lim​−x21​x1​​ =x→0+lim​(−x)=0.​

As such, we conclude that lim⁡x→0+ln⁡x1x=0\displaystyle\lim_{x\to0^+}\frac{\ln x}{\frac{1}{x}} = 0x→0+lim​x1​lnx​=0.

Evaluate lim⁡x→π25cos⁡(−7x)sec⁡(9x)\displaystyle\lim_{x\to\frac{\pi}{2}}5\cos(-7x)\sec(9x)x→2π​lim​5cos(−7x)sec(9x).

Here, we have that:
lim⁡x→π25cos⁡(−7x)sec⁡(9x)=lim⁡x→π2(5)lim⁡x→π2cos⁡(−7x)lim⁡x→π2sec⁡(9x)=5⋅0⋅∞.\lim_{x\to\frac{\pi}{2}}5\cos(-7x)\sec(9x) = \lim_{x\to\frac{\pi}{2}}(5)\lim_{x\to\frac{\pi}{2}}\cos(-7x)\lim_{x\to\frac{\pi}{2}}\sec(9x) \\ = 5 \cdot 0 \cdot \infty.x→2π​lim​5cos(−7x)sec(9x)=x→2π​lim​(5)x→2π​lim​cos(−7x)x→2π​lim​sec(9x)=5⋅0⋅∞.

One way we can manipulate the expression is writing sec⁡9x\sec9xsec9x as 1cos⁡9x\frac{1}{\cos9x}cos9x1​.
lim⁡x→π25cos⁡(−7x)sec⁡(9x)=lim⁡x→π25cos⁡(−7x)⋅1cos⁡9x=lim⁡x→π25cos⁡(−7x)cos⁡9x=5⋅11=5\begin{aligned} \lim_{x\to\frac{\pi}{2}}5\cos(-7x)\sec(9x) &= \lim_{x\to\frac{\pi}{2}}5\cos(-7x)\cdot\frac{1}{\cos 9x} \\[1em] &= \lim_{x\to\frac{\pi}{2}}\frac{5\cos(-7x)}{\cos9x} \\[1em] &=\frac{5\cdot1}{1} \\[1em] &= 5 \end{aligned}x→2π​lim​5cos(−7x)sec(9x)​=x→2π​lim​5cos(−7x)⋅cos9x1​=x→2π​lim​cos9x5cos(−7x)​=15⋅1​=5​

And so, we have that lim⁡x→π25cos⁡(−7x)sec⁡(9x)=5\displaystyle\lim_{x\to\frac{\pi}{2}}5\cos(-7x)\sec(9x) = 5x→2π​lim​5cos(−7x)sec(9x)=5.

“The limit does not exist!”

Compute lim⁡x→0ln⁡(1−x)−sin⁡x1−cos⁡2x\displaystyle\lim_{x\to0}\frac{\ln(1-x)-\sin x}{1-\cos^2x}x→0lim​1−cos2xln(1−x)−sinx​.

Using the trig identity cos⁡2+sin⁡2=1\cos^2+\sin^2=1cos2+sin2=1, we can rewrite the limit.
lim⁡x→0ln⁡(1−x)−sin⁡x1−cos⁡2x=lim⁡x→0ln⁡(1−x)−sin⁡xsin⁡2x\begin{aligned} \lim_{x\to0}\frac{\ln(1-x)-\sin x}{1-\cos^2x} &= \lim_{x\to0}\frac{\ln(1-x)-\sin x}{\sin^2x} \end{aligned}x→0lim​1−cos2xln(1−x)−sinx​​=x→0lim​sin2xln(1−x)−sinx​​

Let’s try to find the limit by direct substitution.
lim⁡x→0ln⁡(1−x)−sin⁡xsin⁡2x=ln⁡(1−0)−sin⁡0sin⁡20=00\begin{aligned} \lim_{x\to0}\frac{\ln(1-x)-\sin x}{\sin^2x} &= \frac{\ln(1-0)-\sin 0}{\sin^20} \\[1em] &= \frac{0}{0} \end{aligned}x→0lim​sin2xln(1−x)−sinx​​=sin20ln(1−0)−sin0​=00​​

We get the indeterminate form 0/00/00/0, so we can use L’Hôpital’s rule.
lim⁡x→0ln⁡(1−x)−sin⁡xsin⁡2x=lim⁡x→0ddx(ln⁡(1−x)−sin⁡x)ddx(sin⁡2x)=lim⁡x→0−11−x−cos⁡x2sin⁡xcos⁡x\begin{aligned} \lim_{x\to0}\frac{\ln(1-x)-\sin x}{\sin^2x} &= \lim_{x\to0}\frac{\diff{}{x}(\ln(1-x)-\sin x)}{\diff{}{x}(\sin^2x)} \\[2em] &= \lim_{x\to0}\frac{\frac{-1}{1-x}-\cos x}{2\sin x\cos x} \end{aligned}x→0lim​sin2xln(1−x)−sinx​​=x→0lim​dxd​(sin2x)dxd​(ln(1−x)−sinx)​=x→0lim​2sinxcosx1−x−1​−cosx​​

Then, using the identity sin⁡(2x)=2sin⁡xcos⁡x\sin(2x)=2\sin x\cos xsin(2x)=2sinxcosx, we rewrite the limit again.
lim⁡x→0−11−x−cos⁡x2sin⁡xcos⁡x=lim⁡x→0−11−x−cos⁡xsin⁡(2x)\begin{aligned} \lim_{x\to0}\frac{\frac{-1}{1-x}-\cos x}{2\sin x\cos x} &= \lim_{x\to0}\frac{\frac{-1}{1-x}-\cos x}{\sin(2x)} \end{aligned}x→0lim​2sinxcosx1−x−1​−cosx​​=x→0lim​sin(2x)1−x−1​−cosx​​

Using limit laws, we evaluate the limit of the numerator and denominator separately.
lim⁡x→0−11−x−cos⁡xsin⁡(2x)=lim⁡x→0(−11−x−cos⁡x)⋅lim⁡x→01sin⁡(2x)\lim_{x\to0}\frac{\frac{-1}{1-x}-\cos x}{\sin(2x)} = \lim_{x\to0}\(\frac{-1}{1-x}-\cos x\) \cdot \lim_{x\to0}\frac{1}{\sin(2x)}x→0lim​sin(2x)1−x−1​−cosx​=x→0lim​(1−x−1​−cosx)⋅x→0lim​sin(2x)1​
