4.3 — Maxima and Minima

The maximum and minimum of a function represent the largest and smallest values the function will ever equal. Today, we’ll examine several types of maxima and minima.

Absolute extrema

Let fff be a function defined over an interval III, and let ccc be a number such that c∈Ic \in Ic∈I.

Absolute maximum

We say that fff has an absolute maximum on III at ccc if f(c)≥f(x)f(c) \geq f(x)f(c)≥f(x) for all x∈Ix \in Ix∈I.

Absolute minimum

We say that fff has an absolute minimum on III at ccc if f(c)≤f(x)f(c) \leq f(x)f(c)≤f(x) for all x∈Ix \in Ix∈I.

Absolute extremum

We say that fff has an absolute extremum on III at ccc if fff has an absolute maximum on III at ccc or an absolute minimum on III at ccc.

A function may have both an absolute maximum and an absolute minimum, just one extremum, or neither.

Three of the examples above had just one extremum. This is due to the fact that those three functions, fff, ggg, and hhh, were unbounded, meaning that their domains were (−∞,∞)(-\infty, \infty)(−∞,∞). However, the following theorem, called the Extreme Value Theorem, guarantees that a continuous function fff over a closed, bounded interval [a,b][a, b][a,b] has both an absolute maximum and an absolute minimum.

Extreme value theorem

If fff is a continuous function over the close, bounded interval [a,b][a,b][a,b], then there exists a point in [a,b][a,b][a,b] at which fff has an absolute minimum and absolute minimum.

Local extrema and critical points

Let fff be a function where III is an open interval in the domain of fff, and let ccc be a number such that c∈Ic \in Ic∈I.

Local maximum

We say that fff has a local maximum at x=cx = cx=c if f(c)≥f(x)f(c) \geq f(x)f(c)≥f(x) for all x∈Ix \in Ix∈I.

Local minimum

We say that fff has a local minimum at x=cx = cx=c if f(c)≤f(x)f(c) \leq f(x)f(c)≤f(x) for all x∈Ix \in Ix∈I.

Local extremum

We say that fff has a local extremum at ccc if fff has a local maximum at ccc or local minimum at ccc.

An interior point is a point on the graph of fff such that it is not an endpoint of fff.

Locating absolute extrema
